# Agent Conversations

This directory stores important conversations between the user and AI agents. These files provide historical context for future agents and developers to understand the rationale behind decisions and changes.

## Naming Convention
Files should be named using the following format:
`YYYY-MM-DD-topic.md`

Example: `2024-10-27-auth-implementation.md`

## Usage
- Save significant discussions, planning sessions, and debugging logs here.
- Use `markdown` format.
- Future agents should read relevant files in this directory to understand previous context.
